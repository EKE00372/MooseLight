# MooseLight

Adjust gamma when player enter some zones and dungeons.

* Dungeons: 
	* Kings' Rest
	* Siege of Boralus
	* Temple of Sethraliss
	* Waycrest Manor
	* Tol Dagor
* Raid:
	* Uldir, Plague Vault
* Other:
	* Vision of Orgrimmar
	* Vision of Stormwind
	* Drustvar, Tiragarde Sound and Stormsong Valley at night.
